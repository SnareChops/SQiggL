Welcome to SQiggL

What is SQiggL?

SQiggL is a SQL template based language that allows developers, DBAs, Data Entry, and any others to script out SQL queries, stored procedure, or scripts with variables that will modify the outputted SQL after parsed.

Why?

  • Some people hate or don't trust ORMs
  • Some developers can't use them due to management
  • You may not be able to use an ORM due to not working in a programming language, for instance you are simply writing a script to be ran manually, but depending on some rule it may need to be slightly different in different scenarios
